# Hotel Reservation System

## Overview

The Hotel Reservation System is a Java-based console application that allows users to manage hotel room bookings efficiently. The system enables customers to search available rooms, book rooms, cancel reservations, view booking details, and simulate payments. Booking records are stored using File I/O for better data management.

## Features

* Search available rooms
* Book hotel rooms
* Cancel reservations
* View booking details
* Payment simulation
* Room categorization (Standard, Deluxe, Suite)
* File handling for storing booking records
* Object-Oriented Programming (OOP) implementation

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* File I/O
* Console-Based User Interface

## How to Run

1. Open the project in VS Code or any Java IDE.

2. Compile all Java files:

   javac *.java

3. Run the main file:

   java HotelReservationSystem
